# Counter App

A simple JavaScript counter application built with HTML, CSS, and vanilla JS.

## Features
- Increment / decrement counter
- +5 / -5 step buttons
- Reset counter
- Toggle to allow/disallow negative numbers
- Color changes based on value (positive, zero, negative)
- Smooth hover and active button effects

## Tech Used
- HTML
- CSS (Flexbox, transitions)
- JavaScript (DOM manipulation, event listeners)

UI polished, logic complete. Ready to view live on GitHub Pages.
